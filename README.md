# ✨ Glassmorphism UI Buttons

A modern UI project featuring animated gradient buttons, ripple effects, and glassmorphism cards.

## 🚀 Features
- 🧊 Glassmorphism cards
- 🌈 Animated gradient buttons
- 🌊 Ripple click effect (Material style)
- 💥 Bubble animation
- 📱 Fully responsive grid
- 🎨 Multiple color variants
